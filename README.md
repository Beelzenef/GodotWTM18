# Taller de Godot

Un proyecto de [Godot Engine](https://godotengine.org/) para los talleres en diferentes eventos:

* [Women Techmakers 2018](https://www.meetup.com/es-ES/Google-Developer-Group-Malaga/events/246863622/)
* [devFest2018 Málaga](https://www.meetup.com/es-ES/Google-Developer-Group-Malaga/events/254547379/)
* [OpenSouthCode 2019](https://www.opensouthcode.org/conferences/opensouthcode2019/program/proposals/253)

Encuentra las diapositivas en [este enlace](https://docs.google.com/presentation/d/1Az4wJAQjbiHxtmja0APHB8z6ugfVHULyb2yiHz_zJLE/edit?usp=sharing).
